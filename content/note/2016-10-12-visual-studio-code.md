## Key Bindings for Visual Studio Code

- Ctrl+Shift+P: Command Palette
- Ctrl+P: Go to file
- Alt+Up: Move lines up
- Alt+Down: Move lines down
- Ctrl+Alt+Up: Add cursors above
- Ctrl+Alt+Down: Add cursors below
- Ctrl+`: Toggle terminal
